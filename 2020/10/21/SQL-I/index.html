Select conditions

  • DISTINCT to remove the duplicate data
    DISTINCT(name)

  • LEFT to select the left first characters

  • LEFT(description, 20) is to find the first 20 characters

  • RIGHT to select the last right characters

  • LEN to calculate the length of the

  • CARINDEX to find the specific characters
    CARINDEX('@', url)

  • SUBSTRING to subset the specfic length of character
    SUBSTRING(email, 5, 7) means from the 5th character of email to extract 7 characters.

  • REPLACE to replace ‘a’ with ‘b’
    REPLACE(email, '@', '#') is to replace @ with #

WHERE

NOTE: Where can only be applied before GROUP by

  • LIKE LIKE 's%'

(To select row begin with ‘s’)

  • IN WHERE 'daniel' IN employee
  • NOT
  • BETWEEN | NOT BETWEEN
    BETWEEN 200 and 500
    (To select the value between 200 and 500)
  • IS NULL | IS NOT NULL
    TO select NULL value

    Order (ORDER BY)

  • ORDER BY is used to order the retreived data
  1. alphabetical order order. (eg. ORDER BY name)
  2. column number order. (eg. ORDER BY 2 sorted according to column 2)
  3. Order is default as ascending, but it can change the order by specifying DESC (eg. ORDER BY DESC)
